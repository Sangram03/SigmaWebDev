Block Elements (Most Commonly Used First)

<div>: A generic container for flow content.
<p> Paragraph.
<h1>,<h2>,<h3>,<h4>,<h5>,<h6>,<h7>  : Headings.
<ul> : Unordered list
<ol> : Ordered list.
<li> : List item.
<form> : A section containg from Heading .
<table> : table
<section> : A standalone section of a document.
<header> : A container for introductory content or a set of navigation links.
<footer> : Footer of a section or page.
<nav> : A section of a page that contains navigation links.
<article> : A self-container composition in a document.
<aside> : A section of a page that contains information indirectly related to the main
<canvas> : A container used to draw graphics via JavaScript.
<dl> : Description list.
<dt> : Term in a description list .
<dd> : Description in a description list .
<figure> :Any content that is referenced from the main content.
<figcaption> : A caption for a <figure> element.
<address> : Contact information for the author or owner of the document.
<hr> : A thematic break or a horizontal rule.
<tfoot>: Footer of a table.



Inline Elements (Most Commonly Used First)
<a> : Anchor or  hyperlink .
<img> : Image.
<span> : Generic Inline container.
<input> : Input field.
<lebel> : lebel for a form element.
<strong> : Strong emphasis.
<em> : emphasized text.
<br> : Line break.
<code> : Code snippet.
<b> :Bold text.
<i> : Italic text.
<u> : Underline text.
<small> : Smaller text.
<sub> : Subscript.
<sup> : Superscript.
<mark> :Marked or highlighted text.
<q>; Short inline quotation.
<cite> : Citation.
<kbd> : Keyword input.
<samp> : Sample output.
<var> :variable in a mathematical expression or programming context.
<time> :Time.
<abbr> : Abbreviation.
<data> : Machine-readable translation of content.
<acronym> : Acroym (Not support in HTML5).
